Event Example

It is possible to track events triggered on elements matching a selector. In this example we track the click event on all elements that match the selector "a#submit".

y.set("_elem", ["a#submit", "click"]);

Click me

By clicking on the link above, a tracking event will be fired before the browser follows the link. To see this open the developer tools and navigate to the network tab. Check the "preserve logs" checkbox. You should see a tracking call being fired before navigating to a new page.
