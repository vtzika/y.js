Advanced A/B Testing

In this example it is demonstrated how you can create advanced control groups of your users

y.set("_ab", [{x: 1, y: 5, z: 0.5}]);

Using this setup y.js will create three control groups (x, y and z) which have a probability distribution of 1, 5 and 0.5 respectively. The probability of each group affects the chances of the particular group to be randomly selected.

If "_ab" was not explicitly set, it will default to {a: 0.5, b: 0.5}.
